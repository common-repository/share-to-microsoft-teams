== Description ==
 
WordPress websites can use the launcher script to embed Share to Teams buttons on their webpages which will launch the Share to Teams experience in a popup window when clicked. 
This will allow you to share a link directly to any person or Microsoft Teams channel without switching context.
 
== Installation ==
 
1. Upload content on the ms-teams-share.zip file to /wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Refresh your wordpress site and a new Microsoft Teams Share button will be added to the side of the site 

== Frequently Asked Questions == 

Does it require a Microsoft Teams account?
Yes in order to share a post on Teams you need to be autenticated with your account on Microsoft Teams.
